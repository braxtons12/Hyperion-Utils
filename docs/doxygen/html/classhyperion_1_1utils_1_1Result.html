template<Passable T, ErrorType E>
hyperion::utils::Result class

Represents the result of an operation that can fail. Every Result() is either Ok, indicating success and containing a value or Err, indicating failure and containing an Error type.

Template parameters
T - the type contained in the case of success
E - the Error type contained in the case of failure
