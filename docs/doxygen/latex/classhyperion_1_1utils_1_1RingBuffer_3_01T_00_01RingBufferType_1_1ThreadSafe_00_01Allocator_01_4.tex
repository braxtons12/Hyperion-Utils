\subsubsection*{template$<$Default\+Constructible T, template$<$ typename Element\+Type $>$ typename Allocator$>$\newline
class hyperion\+::utils\+::\+Ring\+Buffer$<$ T, Ring\+Buffer\+Type\+::\+Thread\+Safe, Allocator $>$}

A simple Ring Buffer implementation. Supports resizing, writing, reading, erasing, and provides mutable and immutable random access iterators. 

\hypertarget{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_autotoc_md8}{}\doxysubsection{Iterator Invalidation}\label{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_autotoc_md8}

\begin{DoxyItemize}
\item Iterators are lazily evaluated, so will only ever be invalidated at their current state. Performing any mutating operation (mutating the iterator, not the underlying data) on them will re-\/sync them with their associated {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer}{Ring\+Buffer}}}. The following operations will invalidate an iterator\textquotesingle{}s current state\+:
\item Read-\/only operations\+: never
\item clear\+: always
\item reserve\+: only if the {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer}{Ring\+Buffer}}} changed capacity
\item erase\+: Erased elements and all following elements
\item push\+\_\+back, emplace\+\_\+back\+: only {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_ae94ac0c9e57c971a96ee45a48d1c929b}{end()}}} until {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_aecd1607ff19990b5b34dca3b9dd62eb1}{capacity()}}} is reached, then {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_a149b48da57e072f8e4dc5710637c9d70}{begin()}}} and {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_ae94ac0c9e57c971a96ee45a48d1c929b}{end()}}}
\item insert, emplace\+: only the element at the position inserted/emplaced
\item pop\+\_\+back\+: the element removed and {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_ae94ac0c9e57c971a96ee45a48d1c929b}{end()}}}
\item pop\+\_\+front\+: the element removed and {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer_3_01T_00_01RingBufferType_1_1ThreadSafe_00_01Allocator_01_4_a149b48da57e072f8e4dc5710637c9d70}{begin()}}}
\end{DoxyItemize}
