\doxysubsection{Detailed Description}
This is a simple change-\/of-\/value detector. 

Compile-\/time configurable Policy-\/based ring buffer.

Hyperion \mbox{\hyperlink{classhyperion_1_1utils_1_1Option}{Option}} and Hyperion \mbox{\hyperlink{classhyperion_1_1utils_1_1Result}{Result}}.

Hyperion logging facilities.

This is a Lock-\/\+Free Single-\/ended Queue implementation using contiguous memory allocations.

Ignore is used to ignore an arbitrary number of unused parameters.

\mbox{\hyperlink{classhyperion_1_1utils_1_1Error}{Error}} is the base type for communicating recoverable errors inside Hyperion.

This is useful for when you need to store a value and track whether writes to the value are actually changes

\mbox{\hyperlink{classhyperion_1_1utils_1_1Error}{Error}} speaks {\ttfamily std\+::error\+\_\+code} as well as custom error messages, and combined with {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1Result}{Result}}} is easily composable and propogated

Hyperion\textquotesingle{}s logging facilities are robust and composable. Behavioral (Policy) configuration is configurable at compile time with via template parameters, and output configuration is configurable by supplying the desired {\ttfamily Sink}s on creation


\begin{DoxyItemize}
\item {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1Option}{Option}}} is a port of Rust\textquotesingle{}s {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1Option}{Option}}} type to C++, and as such as a replacement to {\ttfamily std\+::optional} with various monadic operations and an improved interface and A\+PI
\item \textquotesingle{}\mbox{\hyperlink{classhyperion_1_1utils_1_1Result}{Result}}\textquotesingle{} is a port of Rust\textquotesingle{}s {\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1Result}{Result}}} type to C++, and as such is an alternative to the proposed {\ttfamily std\+::expected} or {\ttfamily boost\+::outcome}
\end{DoxyItemize}

{\ttfamily \mbox{\hyperlink{classhyperion_1_1utils_1_1RingBuffer}{Ring\+Buffer}}} has an A\+PI matching {\ttfamily std\+::vector} in its default configuration or can be configured to provide a subset of that A\+PI, but provide lock-\/free concurrency
