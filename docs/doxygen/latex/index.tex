 Hyperion\+Utils is a header-\/only C++20 library of utilities, data-\/structures, and monads used in the Hyperion game engine.~\newline
 It is currently under active development and has not yet hit a release point.

Hyperion\+Utils is not dependent on the engine and can be used freely separately from any other Hyperion projects.

Some of the features of Hyperion\+Utils include\+:


\begin{DoxyItemize}
\item Robust, Compile-\/time configurable logging (in progress)
\item Datastructures, such as queues supporting lock-\/free concurrency (initial implementation complete; A\+PI unlikely to change, implementation subject to change)
\item Error handling facilities similar to Rust and boost\+::outcome (initial implementation complete; A\+PI unlikely to change, implementation subject to change)
\item Option monad based on Rust\textquotesingle{}s Option (complete)
\item Concepts, Type Traits, and meta-\/programming functions (in progress; Those already implemented unlikely to change)
\item Rust-\/style owning synchronization types (in progress)
\end{DoxyItemize}\hypertarget{index_autotoc_md1}{}\doxysection{Getting Started}\label{index_autotoc_md1}

Tests are currently setup as an isolated project in the \char`\"{}test\char`\"{} subdirectory.~\newline
 This prevents collisions between other googletest builds in Hyperion\textquotesingle{}s other sublibraries.~\newline
 To run the tests, simply configure and build the test project, then run the resulting \char`\"{}\+Test\char`\"{} executable\+:~\newline
